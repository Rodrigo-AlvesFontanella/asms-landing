# Plano comercial — base de landing para a Beth

## Posicionamento

Landing institucional para **assessoria e consultoria voltada a clínicas médicas e consultórios**.

## Objetivo

- passar autoridade
- apresentar serviços
- preparar terreno para conversas com clínicas, consultórios, parceiros e convênios

## Estrutura sugerida

1. Hero com proposta de valor
2. Sobre a consultoria
3. Metodologia de atuação
4. Serviços
5. Diferenciais
6. CTA final

## Como vender essa entrega

Você pode apresentar isso como:

- diagnóstico de posicionamento
- direção visual institucional
- estrutura de copy e autoridade
- implementação da landing
- refinamentos para apresentação comercial

## Reuso

Essa base pode ser adaptada para outros clientes de saúde alterando:

- headline
- subtítulo
- blocos de serviço
- prova social
- CTA
