# Kit de Prompts — Beth (consultoria para clínicas e consultórios)

Use estes prompts no Cline para acelerar briefing, direção visual, arquitetura de página e refinamento comercial.

## 1. Briefing estratégico

```text
Quero estruturar uma landing para uma consultoria/assessoria voltada a clínicas médicas e consultórios.

Antes de propor layout, faça um briefing guiado curto:
- identifique o objetivo comercial da página
- identifique o público decisor
- levante dores, diferenciais e provas de autoridade necessárias
- faça perguntas uma por vez, de forma prática
```

## 2. Direções visuais

```text
Com base no briefing, proponha 3 direções visuais para uma landing institucional de consultoria para clínicas e consultórios.

Para cada direção, explique:
- estilo visual
- sensação transmitida
- prós
- contras
- adequação para atrair clínicas, consultórios, parceiros e convênios

Depois recomende a melhor opção.
```

## 3. Estrutura da landing

```text
Agora monte a arquitetura de conteúdo da landing.

Quero uma estrutura com foco em autoridade e apresentação de serviços para clínicas e consultórios.
Organize em seções e explique a função comercial de cada uma.

Inclua:
- hero
- dores do cliente
- como funciona a consultoria
- serviços
- diferenciais
- prova de confiança
- CTA final
```

## 4. Copy inicial

```text
Escreva a copy inicial da landing com tom institucional, consultivo e premium.

Quero:
- headline
- subheadline
- CTA principal
- textos curtos por seção

Evite exagero publicitário.
A linguagem deve transmitir confiança, especialização no setor de saúde e capacidade de organizar o crescimento da clínica.
```

## 5. Refinamento visual premium

```text
Refine esta landing para parecer mais premium e institucional sem perder clareza comercial.

Melhore:
- hierarquia visual
- espaçamento
- blocos de credibilidade
- contraste e elegância
- percepção de autoridade

Explique primeiro o que vai mudar e por quê.
```

## 6. Revisão comercial final

```text
Faça uma revisão final desta landing como se você estivesse avaliando uma proposta para uma consultoria de clínicas e consultórios.

Analise:
- clareza da proposta de valor
- autoridade percebida
- coerência com público B2B de saúde
- qualidade da copy
- pontos que aumentam ou reduzem valor percebido

Liste melhorias por prioridade.
```

## 7. Prompt para gerar variações de proposta

```text
Crie 3 versões desta landing:
1. mais premium institucional
2. mais comercial corporativa
3. mais humana e consultiva

Mantenha o mesmo serviço base, mas mude narrativa, distribuição de blocos e tom visual.
Explique quando cada versão seria melhor para apresentar ao cliente.
```
